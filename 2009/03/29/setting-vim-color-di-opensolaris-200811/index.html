Setting VIM Color Di OpenSolaris 2008.11

Mar 29th, 2009 1:56 am | Comments

Buat teman-teman yang seneng bermain di Terminal atau Console, pasti sudah tidak asing lagi kan ama yang namanya Vim :) Yups, Vim adalah text editor yang berjalan di mode teks sedangkan di mode GUI temen-teman bisa pakai yang namanya GVim.

Nah yang paling saya sukai di Vim yaitu syntax hightlighting-nya yang sangat bagus dan tiap-tiap file type beda-beda warna-nya :) :D (keren banget). Mungkin teman-teman juga sudah tahu kalau untuk meng-enable pewarnaan kita bisa pakai perintah :syntax on pada mode command di Vim, tapi apa yang terjadi ketika saya mencoba memberikan perintah tersebut di Vim yang berjalan di OpenSolaris ? Hasilnya adalah seperti gambar dibawah ini :
Tampilannya koq cuma bold aja yah ???

dan ketika saya mencoba meng-enable line number dengan memberikan perintah :set nu hasilnya menjadi seperti ini :
Wah tambah parah, koq malah dikasih underline :(

Setelah googling sebentar dan menemukan link disini, akhirnya saya membuat sebuah file .vimrc yang terletak di /export/home/martin/ dan mengisi file .vimrc tersebut dengan konfigurasi seperti dibawah ini :

set term=xtermc
syntax on
set nu

dan hasil akhirnya adalah seperti ini :
Hm.. looking good now :)

Cool kan ?? Untuk konfigurasi yang lain, silahkan ditambahkan sendiri-sendiri sesuai dengan selera masing-masing :)
